## Overview

This is a **demo dashboard** simulating Rocket Doctor AI’s mission to expand access to timely care and improve patient outcomes.  
It demonstrates:  

- **Expanding Access:** Simulate patients in rural, remote, and urban locations  
- **Supporting Continuity:** Track virtual consults, referrals, labs, and follow-ups  
- **Reducing ER Visits:** AI recommendation highlights patients requiring urgent attention  
- **Collaboration:** Centralized dashboard for care teams to monitor multiple patients

---

## Features

- Interactive **dashboard** using Streamlit  
- **Add new patients dynamically** with symptoms and vitals  
- **AI recommendations** based on patient data (Home care, Virtual consult, ER alert)  
- **Trend visualization** for vitals over time  
- Color-coded recommendations for clarity  
- Easy-to-use **real-time simulation** of patient monitoring
